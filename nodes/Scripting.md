# Scripting

---
### Overview

Scripting is the process of writing a series of commands that are executed in a specific order by a program or a scripting engine.
These commands are often written in a human-readable language.
They are useful because they can be used to automate tasks, execute a sequence of commands instead of manually writing each command one-by-one.
