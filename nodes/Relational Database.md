# Relational Database

Relational databases store data in tables with rows and columns, similar to spreadsheets.

---
### Relationship types

1. One-to-One
2. One-To-Many
3. Many-To-Many

These relationships are created using foreign keys that connect rows between tables.

Benefits:
1. Structured data
2. Data integrity
3. Scalable schema
4. Powerful querying

Popular DBs:
- MySQL
- PostgreSQL
- SQLite
- Microsoft SQL Server
- MariaDB
