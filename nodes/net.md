# net

`net` is a legacy Windows CLI utility for managing services and network components.

---
### `net start`

It sends a request to the Service Control Manager (SCM) to start a Windows service.
```bash
net start <service_name>
```

### `winnat` - Windows NAT (Network Address Translation)

This service enables NAT functionality in Windows.
