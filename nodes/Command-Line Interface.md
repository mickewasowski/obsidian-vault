# Command-Line Interface

---
### Overview

Command-Line Interface is the *environment or method* of interacting with a system using text commands.


> [!tip] This is what the Terminal opens to access the Command-Line Interface (*the way of interacting* with the system). Command-Line Interpreter is *the program inside the terminal* that processes your commands (like Bash or PowerShell).
