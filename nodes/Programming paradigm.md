# Programming paradigm

A programming paradigm is a fundamental style of programming that defines how developers structure and write code.

---
### What is a Programming paradigm?
A Programming paradigm is a style of writing computer programs.
We can think of it like *different strategies for organazing and expressing logic in software*.

A Programming paradigm defines:
- how programs are structured
- how data and behavior are organized
- how problems are broken down


### Most common paradigms

1. Imperative Programming [[Imperative programming style]]
2. Object-oriented Programming [[Object-oriented Programming]]
3. Functional Programming [[Functional Programming]]
4. Declarative Programming [[Declarative Programming]]
